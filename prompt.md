あなたはプロの音楽教師です。曲名「${songName}」の冒頭1フレーズを以下のJSON形式で出力してください。

{
    "melody": [
    "歌詞の1フレーズ目",
    "ド レ ミ ファ...",
    "歌詞の2フレーズ目",
    "ミ レ ド..."
  ]
}
条件：
1. 歌詞のすぐ下に、そのメロディをカタカナの「ドレミ」で正確に書いてください。
2. ♯（シャープ）や♭（フラット）がある場合は「ド#」のように書いてください。
3. リズムがわかるように、音と音の間はスペースで区切ってください。
4. 1フレーズ（4〜8小節程度）だけで良いので、音程の正確さを最優先してください。
5. もし有名な曲でなくメロディが不明な場合は、適当に作らずに「ERROR」とだけ返してください。

出力例：
かえるのうたが
ド レ ミ ファ ミ レ ド
きこえてくるよ
ミ ファ ソ ラ ソ ファ ミ
